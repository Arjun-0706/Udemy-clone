# 🎓 Udemy Clone (Frontend)

This project is a **Udemy homepage clone** built using **HTML and CSS**.
It recreates the layout of the Udemy website including the navigation bar, course cards, categories, and footer.

The purpose of this project was to practice **frontend layout design, Flexbox, CSS animations, and responsive design**.

---

## 🚀 Live Demo

🔗 https://arjun-0706.github.io/Udemy-clone/

---

## ✨ Features

* Responsive navigation bar
* Search bar with icons
* Course recommendation section
* Popular courses section
* Topic suggestion tags
* Sale banner with animation
* Hover popup for **My Learning**
* Responsive design using media queries

---

## 🛠 Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Animations
* Media Queries
* Google Fonts
* Font Awesome Icons
